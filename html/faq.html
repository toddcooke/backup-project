FAQ / Guide

For questions or extra information, please consult the guide below.

Regular Backup

-- To set up a basic backup, start here. You will be prompted to enter a date you want your data backed up. On that date, every 60 seconds the program will check to see if the backup has already been performed. The values for a date can either be entered manually in, or you can click the arrow located on the right of the field to drop down a calendar and pick your backup date.

-- You will also be asked to enter how many days you would like between your backups. We understand the need to maximize the use of RAM and disk space on your computer, so we have provided the option to backup only as often as you would like. Enter a numeric value that will represent the number of days between backups.

-- Finally, you will be asked to enter the path to the file or directory you would like backed up.

The following gives examples on what a file path looks like in their respective Operating Systems. The examples will show how to backup a user's Pictures directory.
  • Windows: C:\Users\user1\Pictures
  • Mac OSX: /Users/user1/Pictures
  • Linux: /home/user1/Pictures

Manage Backups

-- After configuring your backup dates, you can view all current backups from the "View Backups" link. Here you will find a table that has all relevant information on your backups.

Here's a quick explanation on what the values on the table mean:
  • Bup_ID: This is a value that identifies your backups, starting at 1.
  • Path: The path to the directory you want backed up.
  • Offset: The value that is how many days between backups.
  • Date of first backup: The date the very first backup was performed.
  • Date of performed backup: The date the most recent backup was performed.

Restore Backup

-- If disaster has struck and you need to recover your backup, follow the "Manage Backups" link. Here you will find a table of all backups that you have scheduled, with a blue "Restore" button found on the right side. Locate the backup that you would like to restore, click "Restore", and... that's it!

Delete Backup

-- If there's a newer version of your backup and you'd like to delete the older backup, it's as simple as going to "Manage Backups", finding the backup you'd like to delete, and clicking the red "delete" button.
